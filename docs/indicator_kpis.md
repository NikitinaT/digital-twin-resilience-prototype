# Indicator KPIs

Define lightweight KPIs used in the demo. These are **illustrative**, not claims about real infrastructure.

## Recommended KPI set
- **Energy availability (%):** fraction of usable energy supply for campus loads.
- **Connectivity (%):** usable communication availability (network reachability/latency capacity).
- **Academic continuity (%):** fraction of teaching/research processes that can operate.
- **Critical services (%):** availability of core services (SOC/command, data center, safety systems).

## Notes
- Use ranges where uncertainty is high (e.g., 5–15%).
- Keep the values consistent across slides: S1 should be the low point, S3 shows partial recovery.
